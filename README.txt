===============================
      TO DO LIST - v1.0
===============================

A modern, lightweight To-Do List application for managing your daily tasks with ease.

-------------------------------
FEATURES
-------------------------------
- Add, edit, and delete tasks
- Prioritize tasks by importance
- Mark tasks as completed
- Auto-save feature
- Clean, modern interface

-------------------------------
INSTALLATION
-------------------------------
1. Unzip the folder.
2. Run the file: ToDoList.exe

No installation required. Just run and use!

-------------------------------
SYSTEM REQUIREMENTS
-------------------------------
- Windows 10 or higher (64-bit)
- .NET Framework 4.7+ (if required)
- 50 MB disk space

-------------------------------
USAGE INSTRUCTIONS
-------------------------------
- Double-click `ToDoList.exe` to launch the app.
- Write all you important tasks
- Then click on Add task
- Select a task
- Click on "Remove Selected Task" to remove it
- You can switch between Dark and Light mode by clicking on "Toggle Dark Mode"
